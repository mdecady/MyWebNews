# Проект "The Newspaper" 

Этот репозиторий содержит кастомную тему для CMS WordPress и модули, разработанные в ходе курса.

## Особенности проекта:
- **Дизайн:** Полностью авторская верстка на Bootstrap 5 с соблюдением сетки (1 большая новость + 2 маленькие).
- **Интерактив:** Реализован слайдер новостей на чистом JavaScript (Vanilla JS) без использования сторонних плагинов.
- **Backend:** Динамический вывод контента из БД MySQL с использованием PHP.
- **Собственный модуль:** Разработан модуль "Фотоконкурс", включающий:
  - Форму загрузки изображений с валидацией.
  - Галерею с системой голосования (лайков).
  - Модерацию заявок через базу данных.

## Технологии:
HTML5, CSS3, JavaScript (ES6), PHP 8, SQL, WordPress API.

## Структура:
- `/Theme/` — исходный код темы и шаблонов страниц.
- `/Database/` — дамп базы данных со структурой новостей и конкурса.
